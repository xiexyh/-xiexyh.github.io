# 课堂笔记

# Query 对象 查询数据库

  • 查多条记录
    方法： list()
    语法： Query q = session.createQuery("from 类名");
    注意： from 要跟类的名称，不跟表的名称。
    ** 例：** 查询 Dept 表中 Sales 部的信息
  • 查一条记录
    方法： uniqueResut()
    语法： Query.uniqueResult()
    功能：从表中查出一条记录
    ** 例：** 查询 emp 表中 empno 等于 7499 的记录
  • 只查部分列（投影查询）
  • 方法 1：
    方法： session.createQuery(select 列1,列2,…. from 类名)
    效果：查询出来的每条记录被放在一个 Object [] 中
    ** 例：**1 条记录放在一个 obj [] 数组中存放
  • 方法 2：
    首先，在对应类中添加构造方法，为要查询的列赋值
    然后使用方法： session.createQuery(select new 类名(列1,列2,…) from 类名)
    效果：查询出来的每条记录就是一个 Java 对象
    ** 例：** 以 Java 对象方式返回结果
  • Where 条件查询
    ** 例：** 查询 emp 表中，ename 是 Smith 的用户

注： 当查询条件是参数，则需要使用占位符或者参数名进行绑定
** 例 1：** 使用占位符？设置参数值

** 例 2：** 使用参数名设置参数值

指定参数类型：
setBoolean (参数名，参数值)
setByte (参数名，参数值)
setDouble (参数名，参数值)
setDate (参数名，参数值)
setString (参数名，参数值)

  • 分页查询
    语法：
    从第几条开始显示: Query.setFirstResult(m)
    总共显示几条: Query.setMaxResults(n)
    功能：显示结果中的第 m 条开始的 n 条记录
    ** 例：** 查询 emp 中按 sal 降序排列第 5 到第 9 条记录

# Criteria 对象（多条件查询）

语法： Criteria c=session.createCriteria()

  • 显示所有记录
    ** 例：** 显示 emp 中的所有记录
  • 添加限制条件
    ** 例：** 查询 job 为 CLERK、sal>1000、入职时间在 1981-04-01 到 1985-09-09 之前的员工

# 反向工程

功能：从数据库中的表 自动生成 Java 类、类 - 表映射文件
步骤：
（1）在 Package Explorer 中 左键点击工程

（2）MyEclipse-->Project Capabilities-->Add Hibernate Capacilities

（3）选择 Hibernate 版本

（4）选择数据库配置文件

（5）创建 HibernateSessionFactory 类


（6）在 MyEclipse 中打开 Database Explorer

（7）新建一个连接


数据库驱动程序名： oracle.jdbc.driver.OracleDriver
数据库实例： jdbc:oracle:thin:@主机名/IP:1521:数据库实例名
（8）在 DB Explorer 中登录数据库

（9）在 DB Browser 中找到要生成 PO 类、映射文件的表

（10）选择生成的 PO 类、映射文件放置的目录

（11）查看生的 PO 类、映射文件

（12）说明：
自动生成的 PO 类的数据类型，有时需要根据实际情况进行修改。
（例如：java.lang.BigDecimal 修改成 java.lang.Long）
