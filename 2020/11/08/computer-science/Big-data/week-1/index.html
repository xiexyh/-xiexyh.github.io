# 关闭防火墙

关闭防火墙： systemctl stop firewalld.service

# 配置 ssh 免密登录

  • 1. 更新 SSH sudo apt-get install openssh-server
    在安装 SSH 过程中，如果提示问 “Do you want to continue?”，请输入 “Y”。
    安装好后，在终端中输入 “ssh localhost” 需要输入密码才能访问。
  • 2. 配置 SSH
    (1)、进入文件 cd ~/.ssh/ # 若没有该目录，请先执行一次 ssh localhost
    (2)、生成公钥 ssh-keygen -t rsa # 会有提示，都按回车即可
    (3)、授权 cat ./id_rsa.pub >> ./authorized_keys # 加入授权
    再次，在终端中输入 “ssh localhost” 无需密码直接登录，输入 exit 可以退出 ssh 登录。
  • 3. 远程发送公钥同步到 haddop2 和 hadoop3 上
    scp ~/.ssh/id_rsa.pub root@hadoop2:~/.ssh/
    scp ~/.ssh/id_rsa.pub root@hadoop3:~/.ssh/
  • 4. 依次进入 hadoop2、hadoop3 中进行授权操作 cat ./id_rsa.pub >> ./authorized_keys

# 安装 JDK

  • 1. 官网下载 jdk:https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html
  • 2. 使用 tar 解压文件到 /opt 目录在 tar -zxvf jdk-8u161-linux-x64.tar.gz -C /opt/
  • 3. 查看是否安装成功 ./java -version
  • 4. 配置环境变量 PATH vim etc/ptofile
    在最末尾添加四行
  • 5. 使环境变量立刻生效 source /etc/profile
  • 6. 同步 hadoop1 主节点的配置到 hadoop2 和 hadoop3 上
    scp -r /opt/jdk1.8.0_162 root@hadoop2:/opt/
    scp -r /opt/jdk1.8.0_162 root@hadoop3:/opt/
    scp -r /etc/profile root@hadoop2:/etc/profile
    scp -r /etc/profile root@hadoop3:/etc/profile
  • 7. 依次到 hadoop2 和 hadoop3 上生效环境变量 source /etc/profile
