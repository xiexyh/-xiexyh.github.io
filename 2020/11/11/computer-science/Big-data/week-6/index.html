# hbase 数据定义（DDL）操作

  • 创建新表
    使用 create 命令来创建一个新的表。在创建的时候，必须指定表名和列族名。
    命令： create 'user', 'info '
  • 列举表信息
    使用 list 命令，列出所有的表信息
    命令： list 'user'
  • 获取表描述
    使用 describe 命令，显示表的相关信息
    命令： describe 'user'
  • 检查表是否存在
    使用 exists 命令，判断表是否存在
    命令： exists 'user'
  • 删除表
    删除表之前，先 disable 表，再使用 drop 命令实现删除表的功能
    命令： drop 'user

# 数据管理（DML）操作

  • 向表中插入数据
    使用 put 命令，将数据插入表中
    命令： put 'user', 'row1', 'info:a', 'value1'
  • 一次性扫描全表数据
    一种获取 HBase 数据的方法是扫描，使用 scan 命令来扫描表的数据。（可以限制扫描的范围）
    命令： scan 'user'
  • 获取一个行数据
    使用 get 命令来获得某一行的数据
    命令： get 'user', 'row1'
  • 禁用一个表
    如果你想要删除一个表或是修改它的设置，或者是其它的情况，都需要首先禁用该表。使用 disable 命令禁用表，enable 命令重新启用表。
    禁用表命令： disable 'user'
    重新启用表命令： enable 'user'
  • 删除数据
    • 删除行中的某个列值
      语法： delete <table>, <rowkey>,  <family:column> , <timestamp>
      命令： delete 'user','rowkey001','info:a'
    • 删除行
      语法： deleteall <table>, <rowkey>,  <family:column> , <timestamp>
      命令： deleteall 'user','row1'
    • 删除表中的所有数据
      语法： truncate <table>
      命令： truncate 'user'
