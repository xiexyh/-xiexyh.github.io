# HDFS 基础命令

# -mkdir 命令

功能：在 hdfs 上创建文件夹
操作：在 hdfs 上创建名为 test1 的文件夹
在终端执行命令： hdfs dfs -mkdir /test1
操作执行截图：

# -ls 命令

功能：显示目录信息
操作：显示 hdfs 根目录的信息
在终端执行命令： hdfs dfs -ls /
操作执行截图：

# -rm 命令

功能：删除文件或文件夹
注意：如果删除的文件夹下面还有文件就必须先把子文件删除，否则删除不成功
操作：删除根目录下的 test1 文件夹
在终端执行命令： hdfs dfs -rm -r /test1
操作执行截图：

# -mv 命令

功能：在 hdfs 目录中移动文件或重命名文件
操作 1：创建 stu1、stu2 文件夹，并查看创建好的文件夹
在终端执行命令： hdfs dfs -mkdir /stu1
在终端执行命令： hdfs dfs -mkdir /stu2
在终端执行命令： hdfs dfs -ls /
操作 1 执行截图：

操作 2：在 stu1 中创建一个名为 q.txt 的文件夹并查看 stu1 的文件信息
在终端执行命令： hdfs dfs -mkdir /stu1/q.txt
在终端执行命令： hdfs dfs -ls /stu1
操作 2 执行截图：

操作 3：将 stu1 文件夹中的 q.txt 文件夹移动到 stu2 文件夹下，执行后查看 stu1 和 stu2 文件夹下的内容
在终端执行命令： hdfs dfs -mv /stu1/q.txt /stu2
在终端执行命令： hdfs dfs -ls /stu1
在终端执行命令： hdfs dfs -ls /stu2
操作 3 执行截图：

操作 4：将 stu2 下的 q.txt 文件夹重命名为 qqq.txt，执行后查看 stu2 文件夹下的文件信息验证是否重命名成功
在终端执行命令： hdfs dfs -mv /stu2/q.txt /stu2/qqq.txt
在终端执行命令： hdfs dfs -ls /stu2
操作 4 执行截图：

# -cp 命令

功能：从 hdfs 的一个路径复制 hdfs 的另一个路径
操作：将 stu2 文件夹中的 qqq.txt 文件夹复制到 hdfs 根目录下，并查看根目录下是否有 qqq.txt 文件夹
在终端执行命令： hdfs dfs -cp /stu2/qqq.txt /
在终端执行命令： hdfs dfs -ls /
操作执行截图：

# -get 命令

功能：从 hdfs 复制到本地
操作：将 hdfs 上 stu2 文件夹下的 qqq.txt 文件夹复制到 hadoop1 本地的 /opt 目录下，并进入 /opt 下查看是否复制成功
在终端执行命令： hdfs dfs -get /stu2/qqq.txt /opt
在终端执行命令： cd /opt/
在终端执行命令： ls
操作执行截图：

# -put 命令

功能：从本地文件系统中复制文件到 hdfs 文件系统中
操作 1：将本地文件系统中 /root 目录下的 iris.data 文件复制到 hdfs 的 stu1 文件夹下，并查看是否复制成功
在终端执行命令： hdfs dfs -put /root/iris.data /stu1
操作 1 执行截图：

在终端执行命令： hdfs dfs -ls /stu1
操作 2 执行截图：

# -df 命令

功能：统计文件系统的可用空间信息
操作 1：统计 hdfs 文件系统的可用信息
在终端执行命令： hdfs dfs -df -h /
操作 1 执行截图：

# -du 命令

功能：统计一个指定目录下的文件节点数量
操作 1：统计 hdfs 文件系统中 stu1 文件夹下的文件节点数量
在终端执行命令： hdfs dfs -du /stu1
操作 1 执行截图：

# -help 查看帮助命令

功能：输出这个命令的参数
在终端执行命令： hdfs dfs -help/
操作执行截图：

在终端执行命令：hdfs dfs -usage/
操作执行截图：

# 文件权限管理

功能：文件权限管理
操作：将 stu1 文件夹下的 iris.data 文件权限修改为 777，并查看修改结果
在终端执行命令： hdfs dfs -chmod 777 /stu1/iris.data
在终端执行命令： hdfs dfs -ls /
操作执行截图：

# 查看具体文件

功能：查看文件内容
操作：查看 stu1 文件夹下的 iris.data 文件的内容
在终端执行命令： hdfs dfs -cat /stu1/iris.data
操作执行截图：
