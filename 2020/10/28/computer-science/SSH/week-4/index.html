# 课堂笔记

# 表空间

作用：

a. 存放表、视图、索引、触等各种数据库对象。

什么场合需使用表空间

a. 有多个数据库用户（登录名）
为每个用户创建一个表空间，使每个用户能够把自己的表、视图等存放在专有的表空间里，方便管理。
b. 有多个业务系统
例如，某学校的网站上有 “教务管理系统”、“人事管理系统”，可为每个业务系统创建一个表空间，使得每个系统的数据库对象存放在专有的表空间里，方便管理。

# 给用户授权应 遵循的原则

（1）最小分配原则（需要多少权力，就分配多少，绝不多分）

# 常见的 SQL 语句优化

为何需要优化 SQL 语句：提高 SQL 语句的执行效率、缩短执行时间

# SQL 语句优化的通用技巧

  • select 语句中不要使用 ，而要使用具体的列名
    原理：
    只要使用    ，数据库引擎就需要要查询数据字典，从数据字典中得到所有字段。
    不用 *，直接使用具体列名时，数据库引擎就省去了查询数据字典的步骤，从而缩短执行时间。
    所以，确实需要用表中的所有列时采用 *。 其它情况全部使用具体的列名。
  • 当要删除表中的所有记录时，使用 truncate 代替 delete
  • 其它技巧
    使用 Exists 代替 in、驱动表的选择、Where 子句的连接顺序
